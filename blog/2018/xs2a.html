PSD2 XS2A

05 Juni 2018

PSD2

XS2A

To support the EU directive PSD2 the Berlin Group initiative described and specified the process to implement PSD2 in an open and transparent manner.

Adorsys, a development consultancy specialized in the financial and insurance sector made an XS2A blueprint and Java reference implementation of the Berlin Group specification available on GitHub. We created a fork at JavaMoney and contributed some improvements back to the upstream repository by Adorsys via PRs.
