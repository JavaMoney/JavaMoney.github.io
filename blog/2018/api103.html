API Release 1.0.3

18 April 2018

Service Release 3

API Release 1.0.3

A new service release of JSR 354 API was released today.
The major improvement was embracing the Java 9 module system. As usual This patch release 1.0.3 of JSR 354 was be made available on popular Maven repositories.
There are no structural API changes nor JavaDoc, etc. therefore no MR was necessary for this and jcp.org won't be updated. Check the 1.0.3 release tag.

Thanks everyone who helped publish this release.
